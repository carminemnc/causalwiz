# causalwiz: A Package for Causal Inference Analysis

The causalwiz package provides tools for causal inference analysis,
implementing various methods including Inverse Probability Weighting
(IPW) and Augmented Inverse Probability Weighting (AIPW). It includes
utilities for estimating treatment effects, analyzing covariate balance,
and visualizing results.
