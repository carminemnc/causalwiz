# causalwiz: Tools for Causal Inference Analysis

causalwiz provides a comprehensive toolkit for causal inference analysis
in R. The package implements modern methods for estimating treatment
effects, including Inverse Probability Weighting (IPW) and Augmented
Inverse Probability Weighting (AIPW), with built-in diagnostics for
covariate balance and propensity score analysis.
