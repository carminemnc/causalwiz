# causalwiz: Tools for Causal Inference Analysis

causalwiz is a comprehensive toolkit for causal inference analysis in R.
The package provides functions for estimating treatment effects using
various statistical methods, features built-in plotting functions and
utilities designed for causal inference workflows in R
